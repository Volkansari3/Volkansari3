## 🚀 Featured Projects

### 🌌 [Gaia DR3 Stellar Classification](https://github.com/Volkansari3/Gaia-DR3-Stellar-Classification)
> Deep learning on real telescope survey data.

- Stellar classification of **136K Gaia DR3 RVS spectra**
- **1D-CNN** architecture implemented with **JAX/Flax (NNX)**
- End-to-end pipeline: spectra preprocessing → training → evaluation

---

### 🩺 [Medical RAG System](https://github.com/Volkansari3/medical-rag-system)
> RAG pipeline helping Turkish-speaking doctors query English medical literature.

- **PubMed E-utilities** data pipeline with deduplication & rate limiting
- Three retrieval methods compared: **BM25**, **multilingual E5**, **Hybrid RRF**
- Answer generation with **LLaMA-3.1** via Groq, with citation constraints

---

### 📊 [Bank Marketing Prediction](https://github.com/Volkansari3/bank-marketing-streamlit)
> Classic ML, end to end.

- Predicting term deposit subscriptions from marketing data
- Full workflow: EDA → feature engineering → model → **Streamlit** app
